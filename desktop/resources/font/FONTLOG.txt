<a name="cl-2"></a>Avería font family
<a name="cl-3"></a>========================
<a name="cl-4"></a>
<a name="cl-5"></a>
<a name="cl-6"></a>This file provides detailed information on the Avería family of fonts. This
<a name="cl-7"></a>information should be distributed along with the Avería fonts and any
<a name="cl-8"></a>derivative works.
<a name="cl-9"></a>
<a name="cl-10"></a>
<a name="cl-11"></a>Basic Font Information
<a name="cl-12"></a>----------------------
<a name="cl-13"></a>
<a name="cl-14"></a>Avería (&quot;breakdown&quot; or &quot;mechanical damage&quot; in Spanish - related to the root
<a name="cl-15"></a>of the English word &quot;average&quot;) is a Unicode typeface family created from the
<a name="cl-16"></a>average of all fonts on the computer of the creator, Dan Sayers. The process
<a name="cl-17"></a>is described at http://iotic.com/averia/. All metrics are the result of an
<a name="cl-18"></a>averaging process. The included glyphs are those that existed in a majority
<a name="cl-19"></a>of the source fonts - which are the following:
<a name="cl-20"></a>
<a name="cl-21"></a>Range           Description
<a name="cl-22"></a>U+0020-U+007E   Basic Latin
<a name="cl-23"></a>U+00A0-U+00FF   Latin-1 Supplement
<a name="cl-24"></a>
<a name="cl-25"></a>And some glyphs from the following:
<a name="cl-26"></a>
<a name="cl-27"></a>Range           Description
<a name="cl-28"></a>U+0100-U+017F   Latin Extended-A
<a name="cl-29"></a>U+0180-U+021F   Latin Extended-B
<a name="cl-30"></a>U+02B0-U+02FF   Spacing Modifier Letters
<a name="cl-31"></a>U+0370-U+03D5   Greek
<a name="cl-32"></a>U+2000-U+2122   General Punctuation
<a name="cl-33"></a>U+2200-U+25CS   Mathematical Operators
<a name="cl-34"></a>
<a name="cl-35"></a>Avería exists in Regular, Bold, Italic, Bold Italic, Light and Light Italic
<a name="cl-36"></a>styles - as well as a &quot;Gruesa&quot; semi-italic variant made from an average of
<a name="cl-37"></a>all 725 source fonts.
<a name="cl-38"></a>
<a name="cl-39"></a>
<a name="cl-40"></a>ChangeLog
<a name="cl-41"></a>---------
<a name="cl-42"></a>(This should list both major and minor changes, most recent first.)
<a name="cl-43"></a>
<a name="cl-44"></a>02 Nov 2011 (Dan Sayers)  Avería version 1.01
<a name="cl-45"></a>- Fixed bug in generating quotedblbase perimeters
<a name="cl-46"></a>- Slightly changed kerning
<a name="cl-47"></a>
<a name="cl-48"></a>29 Oct 2011 (Dan Sayers)  Avería version 1.00
<a name="cl-49"></a>- First public release
<a name="cl-50"></a>- No hinting is included in this version.
<a name="cl-51"></a>
